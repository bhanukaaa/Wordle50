Welcome to Wordle50


This is a refactored version of my CS50x Final Project, modified to not use the Flask framework or the CS50 libraries,
and to run as a standalone webapp, on GitHub Pages

This whole web app was designed for CS50x's Final Project, inspired by the similarly named problem in Problem Set 2 of CS50x


Wordle is a popular online word-guessing game that challenges players to decipher a hidden word within a set number of attempts.
It's a game of deductive reasoning, vocabulary, and a bit of luck

Upon every guess you make, the letters will be coloured to help you deduce the hidden word
✖
?
✓

A gray coloured cell indicates a letter that does not exist within the hidden word
A yellow coloured cell indicates a letter that is in the hidden word, but isn't in the correct position
A green colored cell indicates a correctly placed letter corresponding to the hidden word

The words used were generated using ChatGPT and other publicly available sources, and I have not personally gone through the thousands of words within the dictionaries running this web app, So apologies in advance for any offensive or nonsensical words included. There is built in error handling to catch any words that aren't the actual word length of the game version being played, as ChatGPT's word list isn't exactly accurate.
